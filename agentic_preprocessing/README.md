# Agentic AI Data Preprocessing

This module implements a non-destructive, human-in-the-loop agentic preprocessing controller for audio datasets.

The agent does not delete raw files. It audits, scores, flags, routes, explains, and recommends preprocessing actions.

Initial target:
- 4 raw human-talk classes
- accepted / rejected / needs_review decision manifest
- dataset quality audit report
- metadata validation
- recommended next actions
