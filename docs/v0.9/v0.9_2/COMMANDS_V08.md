# Commands — agentic_data_preprocessing_v0.8

This file records the reproducible PowerShell commands used for the v0.8-human-corrected-balanced experiment and the later label-aware aggregation analysis.

## 1. Corrected holdout parent mean evaluation

Purpose: official corrected-holdout clip-level evaluation.

```powershell
$HCBBRoot = "human_talk_workspace\tata_v0.8_human_corrected_balanced_pipeline"

$RunDir = "$HCBBRoot\main_models\runs\main_v08_human_corrected_balanced_3exit_20260610_084027"
$CorrectedHoldoutManifest = "$HCBBRoot\corrected_holdout\multilabel_features_manifest_CORRECTED_LABELS.csv"
$HoldoutFeaturesRoot = "human_talk_workspace\tata_v0.6_raw_pipeline\final_holdout_feature_cache\features"
$OutDir = "$HCBBRoot\evaluation\existing_script_parent_mean_fixed"

python scripts\evaluate_tata_final_holdout_parent_level.py `
  --run_dir "$RunDir" `
  --holdout_manifest "$CorrectedHoldoutManifest" `
  --features_root "$HoldoutFeaturesRoot" `
  --out_dir "$OutDir" `
  --threshold_mode fixed_0p5 `
  --aggregation mean `
  --device cpu `
  --batch_size 128
```

Key output:

```text
Exit 3:
Macro-F1     = 0.7801
Micro-F1     = 0.9332
Samples-F1   = 0.9406
Exact Match  = 0.8397
Hamming Loss = 0.0194
```

## 8. Run repeated labelwise aggregation + threshold calibration stability test

Purpose: test whether labelwise aggregation and threshold calibration are stable across repeated 50/50 calibration/evaluation splits.

```powershell
$SegmentPredCsv = "human_talk_workspace\tata_v0.9_labelwise_calibration\verification\v08_parent_mean_fixed\parent_eval_segment_probs_fixed_0p5_mean.csv"
$OutDir = "human_talk_workspace\tata_v0.9_labelwise_calibration\repeated_v07_style_calibration"

New-Item -ItemType Directory -Force -Path $OutDir | Out-Null

python scripts\v0.9\run_labelwise_aggregation_threshold_calibration.py `
  --segment-pred-csv "$SegmentPredCsv" `
  --out-dir "$OutDir" `
  --fixed-threshold 0.5 `
  --seeds 20 `
  --cal-fraction 0.5 `
  --threshold-min 0.10 `
  --threshold-max 0.95 `
  --threshold-step 0.05 `
  --parent-id-col "parent_clip_id" `
  --prob-prefix "exit3_prob_"
```

Key result:

| Method | Macro-F1 mean | Micro-F1 mean | Samples-F1 mean | Exact mean | Hamming Loss ↓ mean |
|---|---:|---:|---:|---:|---:|
| mean fixed thresholds | 0.7802 | 0.9315 | 0.9392 | 0.8371 | 0.0199 |
| **v06 selected aggregation fixed thresholds** | **0.8310** | **0.9345** | **0.9449** | **0.8368** | **0.0193** |
| v07 aggregation + threshold calibrated | 0.8319 | 0.9288 | 0.9363 | 0.8185 | 0.0208 |

Decision: freeze labelwise aggregation with fixed 0.5. Do not use threshold calibration as final because it gives only a tiny Macro-F1 gain but weaker Micro-F1, Samples-F1, Exact Match, and Hamming Loss.
